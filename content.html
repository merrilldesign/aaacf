Image Style Guide

Image Guide

Please refer to the attached Image Size Guide when creating new image files to upload to your site. Images should always be re-sized BEFORE being uploaded via the image manager; do not use the image editor in LiveHTML to edit image sizes.

CSS Classes

Photo caption style
  • imageleft - Aligns an image to the left side of the content area. Content will wrap around the image.
  • imageright - Aligns an image to the right side of the content area. Content will wrap around the image.
  • captionleft - Should be applied to a table containing an image and caption text. This class aligns the table to the left side of the content area and changes the color of the caption text. Content will wrap around the table.
  • captionright - Should be applied to a table containing an image and caption text. This class aligns the table to the right side of the content area and changes the color of the caption text. Content will wrap around the table.
  • largetext - Makes text larger.
  • attribution - Makes the quote attribution smaller when in the quote container.
  • button - Makes a link look like a button.
  • arrowLink - Adds an arrow icon to a link.
  • table - Apply to a data table to make it responsive. The table must have a header row at the top, all headers must be filled in and there cannot be any cells that span multiple rows or columns.
  • noresize - Apply to an image that should not resize with the browser. Please to not apply to any image greater than 200px in width.
  • red - Makes text red. Used only for production notes.

To add an image with a caption:

  1. Add a one column, two row table
  2. Click into the top cell and insert your image
  3. Click into the bottom cell and type/paste the caption
  4. Select the whole table (either by hovering over an edge and selecting it, by right-clicking inside the table and going to "Table Properties", or by, when your cursor is in the table, clicking on the "Table" tag at the bottom of the editor which will select the whole table) and select "captionright" or "captionleft" from the Apply CSS Class menu to float the table to one side and have text wrap around it.

To add a custom container style to a Live HTML module:

  1. Add a LiveHTML module to the page and enter your content. Refer to the container examples on the Design Standards page for formatting.
  2. Click Save & Return to return to the page.
  3. Hover over Settings gear icon and choose "Settings" from menu.
  4. Go to Page Settings tab.
  5. Check the box next to "Display Container?"
  6. From the Module Container dropdown menu, select the Container Style.
  7. Click Update to save your settings.
